## iJasa

### Installation Step

1. bikin file `.env`
2. copy isi `.env.example` ke `.env`
3. `npm install`
4. `npm run dev`
5. `composer install`
6. `php artisan key:generate`
7. ganti `APP_URL` yang di `.env` jadi `https://127.0.0.1:8000/`
8. `php artisan migrate`
9. `php artisan serve`
10. dusk udah diinstal, langsung tes aja pake `php artisan dusk`
11. bobok
